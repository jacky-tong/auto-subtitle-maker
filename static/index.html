核心能力

一个工具，覆盖两种字幕生产场景

🎙️

自动语音识别

基于 OpenAI Whisper 深度学习模型，自动提取视频音轨并生成逐字时间戳，中文识别准确率业界领先。

📝

文稿强制对齐

上传 .docx 或 .txt 文稿，系统通过 Smith-Waterman 对齐算法将文案精确映射到视频时间轴。

🎬

智能断句 + 烧录

智能识别英文单词边界与中文标点断句，杜绝"python"被截断。黑色字幕 + 白色描边，清晰可读。

两种工作模式

根据是否提供文稿，系统自动选择最佳策略

Case A · 有文稿

文稿强制对齐模式

  • 上传视频 + .docx / .txt 文稿
  • 解析文稿为句子序列
  • Smith-Waterman 字符级对齐
  • 字幕文字严格遵循原稿
  • 适合：有文案的解说视频、课程录像
Case B · 无文稿

全自动语音识别模式

  • 仅需上传视频文件
  • Whisper 直接转录语音内容
  • 自动生成时间戳与字幕
  • 无需任何手动编辑
  • 适合：即兴录制、会议记录、vlog

常见问题

支持哪些语言？
默认配置为中文。修改 config.py 中 WHISPER_LANGUAGE 可切换至英文、日语等 99 种语言。中英混合场景也有良好表现。
处理需要多长时间？
取决于视频长度和硬件。CPU 推理约为视频时长的 1~3 倍（tiny 最快，large 最准）。GPU (CUDA) 近乎实时。
字幕样式可以自定义吗？
可以。修改 services/video_service.py 中 burn_subtitles 的 force_style 参数即可调整字体、大小、颜色、描边、位置等。
支持哪些格式？大小限制？
视频：MP4 / MOV / AVI / MKV / WebM。文稿：.docx / .txt。默认上传限制 500MB，可在 config.py 中调整。
